Capítulo 18 Exercícios práticos com o pacote pwr no R

O pacote pwr permite realizar cálculos de tamanho de amostra, poder estatístico e tamanho do efeito para diversos testes estatísticos. Abaixo, seguem exercícios aplicados aos testes t (pareado e não pareado) e Wilcoxon (pareado e não pareado).

Lembre se instalar o pacote pwr

install.packages("pwr")

18.1 Exercício 1: Calcular o tamanho da amostra para um teste t não pareado

Utilize a função pwr.t.test() para determinar o tamanho da amostra necessário para detectar um tamanho de efeito (d) de 0.5, com poder (power) de 0.8 e nível de significância (sig.level) de 0.05, em um teste t bilateral para duas amostras independentes.

library(pwr)
# Tamanho da amostra para teste t não pareado
pwr.t.test(d = 0.5, power = 0.8, sig.level = 0.05, type = "two.sample")

18.2 Exercício 2: Calcular o poder do teste para uma amostra de tamanho fixo (t não pareado)

Considere agora que você dispõe de 26 observações em cada grupo. Calcule o poder estatístico para detectar um tamanho de efeito de 0.5, com nível de significância de 0.05.

# Poder do teste t não pareado para n = 26 por grupo
pwr.t.test(n = 26, d = 0.5, sig.level = 0.05, type = "two.sample")

18.3 Exercício 3: Calcular o tamanho do efeito detectável (t não pareado)

Com uma amostra de 26 observações por grupo, poder de 0.8 e nível de significância de 0.05, qual o menor tamanho de efeito que pode ser detectado?

# Tamanho do efeito detectável
pwr.t.test(n = 26, power = 0.8, sig.level = 0.05, type = "two.sample")

18.4 Exercício 4: Repita os cálculos para outros testes (t pareado, Wilcoxon não pareado/pareado)

18.4.1 a) Teste t pareado

# Tamanho da amostra para teste t pareado
pwr.t.test(d = 0.5, power = 0.8, sig.level = 0.05, type = "paired")

# Poder para n = 26 pares
pwr.t.test(n = 26, d = 0.5, sig.level = 0.05, type = "paired")

# Tamanho do efeito detectável
pwr.t.test(n = 26, power = 0.8, sig.level = 0.05, type = "paired")

18.4.2 b) Wilcoxon não pareado

# Tamanho da amostra para Wilcoxon não pareado (usando pwr.t.test como aproximação)
pwr.t.test(d = 0.5, power = 0.8, sig.level = 0.05, type = "two.sample")

Dica: Para testes não-paramétricos como o Wilcoxon, o pacote pwr não possui uma função específica. Nesses casos, costuma-se usar o cálculo feito para o teste t como uma aproximação. No entanto, recomenda-se aumentar o tamanho da amostra em cerca de 15% (ou seja, multiplicar o valor obtido para o teste t por 1,15), já que testes não-paramétricos geralmente requerem amostras um pouco maiores para alcançar o mesmo poder estatístico. Essa estratégia é especialmente útil para amostras de tamanho moderado ou grande.

18.4.3 c) Wilcoxon pareado

# Tamanho da amostra para Wilcoxon pareado (usando pwr.t.test como aproximação)
pwr.t.test(d = 0.5, power = 0.8, sig.level = 0.05, type = "paired")

O d de Cohen é uma medidas de taamnho de efeito para o teste t. Seu uso não é adequado em testes não-paramétricos como o Wilcoxon (pareado ou não pareado).

18.5 Dica sobre o uso da função pwr.t.test()

A função pwr.t.test() possui os seguintes argumentos principais:

  • n: tamanho da amostra em cada grupo (ou número de pares, para o teste pareado)
  • d: tamanho do efeito (diferença padronizada entre grupos)
  • power: poder estatístico desejado (probabilidade de detectar o efeito, se houver)
  • sig.level: nível de significância (probabilidade de erro tipo I, geralmente 0.05)
  • type: especifica o tipo de teste t: "two.sample" (duas amostras independentes), "paired" (amostras pareadas), ou "one.sample" (amostra única)

Importante:
Deixe como NULL o parâmetro que você deseja calcular. Por exemplo:
- Para calcular o tamanho do efeito: d = NULL - Para calcular o poder: power = NULL - Para calcular o tamanho da amostra: n = NULL

Exemplo prático para calcular o tamanho do efeito detectável:

pwr.t.test(n = 26, d = NULL, power = 0.8, sig.level = 0.05, type = "two.sample")

Neste exemplo, o argumento d fica como NULL, pois é o valor a ser calculado.
Da mesma forma, ajuste o parâmetro que deseja obter nas suas análises.

18.6 Considerações sobre Grupos Desbalanceados

Quando os tamanhos dos grupos são diferentes (desbalanceados), a função pwr.t2n.test() é usada para fazer o cálculo do poder.

18.6.1 Usando pwr.t2n.test()

Esta função permite informar diretamente o tamanho de cada grupo.

# Exemplo: grupo 1 com 95, grupo 2 com 30 observações
library(pwr)
pwr.t2n.test(n1 = 95, n2 = 30, d = 0.5, sig.level = 0.05)
## 
##      t test power calculation 
## 
##              n1 = 95
##              n2 = 30
##               d = 0.5
##       sig.level = 0.05
##           power = 0.6586749
##     alternative = two.sided
  • n1: Tamanho da amostra do primeiro grupo (quantidade de observações no grupo 1).
  • n2: Tamanho da amostra do segundo grupo (quantidade de observações no grupo 2).

Considerações sobre Grupos Desbalanceados

Grupos desbalanceados podem comprometer o poder do teste, aumentar a variância e dificultar a interpretação dos resultados, principalmente se as suposições do teste não forem atendidas. Por isso, sempre que possível, recomenda-se buscar tamanhos de grupos mais equilibrados em estudos experimentais.

18.7 Tamanho do efeito para teste Wilcoxon

O tamanho do efeito é uma medida essencial para complementar a análise estatística, trazendo uma noção da magnitude da diferença entre grupos ou condições. O d de Cohen é uma das medidas de efeito mais populares, especialmente em testes paramétricos como o teste t. Porém, seu uso não é adequado em testes não-paramétricos como o Wilcoxon (pareado ou não pareado).

18.8 Por que NÃO usar o d de Cohen no teste de Wilcoxon?

O d de Cohen é definido como a diferença entre médias dividida pelo desvio padrão combinado dos grupos. Ele pressupõe:

  • Distribuição normal dos dados
  • Comparação direta entre médias e desvios padrão

Por outro lado, o teste de Wilcoxon (tanto o pareado quanto o não pareado/Mann-Whitney) é um teste não paramétrico que compara postos (ranks), e não valores originais. Assim, aplicar a fórmula do d de Cohen em dados tratados por Wilcoxon pode levar a interpretações enganosas, pois:

  • Os postos não têm relação linear direta com os valores originais
  • A magnitude da diferença baseada em ranks não é comparável ao d de Cohen calculado sobre médias
  • Wilcoxon é ideal para dados assimétricos, ordinais ou com outliers, onde o d de Cohen perde sentido

O d de Cohen supõe propriedades dos dados (normalidade, variâncias) que não são garantidas nos contextos em que se aplica o Wilcoxon!

18.9 Medidas de efeito adequadas para o teste de Wilcoxon

As medidas recomendadas são:

  • r de Wilcoxon
  • Cliff’s Delta

18.9.1 Exemplos práticos

18.9.1.1 Teste de Wilcoxon e r de Wilcoxon (não pareado)

# Instalar pacotes se necessário
# install.packages("rstatix")
library(rstatix)

# Dados fictícios
grupo1 <- c(5, 7, 6, 8, 7)
grupo2 <- c(10, 12, 11, 13, 12)

# Teste de Wilcoxon não pareado
wilcox.test(grupo1, grupo2)

# Tamanho de efeito r de Wilcoxon
dados <- data.frame(
  valor = c(grupo1, grupo2),
  grupo = rep(c("A", "B"), times = c(length(grupo1), length(grupo2)))
)
dados %>% wilcox_effsize(valor ~ grupo)

18.9.1.2 Teste de Wilcoxon pareado e r de Wilcoxon

pares1 <- c(10, 12, 15, 14, 11)
pares2 <- c(13, 15, 16, 17, 14)
dados_pareado <- data.frame(
  antes = pares1,
  depois = pares2
)

wilcox.test(dados_pareado$antes, dados_pareado$depois, paired = TRUE)
dados_pareado %>% wilcox_effsize(antes ~ depois, paired = TRUE)

O r de Wilcoxon é uma medida de tamanho de efeito utilizada em testes não paramétricos, como o teste de Wilcoxon para amostras pareadas e o teste de Mann-Whitney (Wilcoxon rank-sum) para amostras independentes. Ele indica a magnitude da diferença entre grupos ou condições, de forma semelhante ao coeficiente de correlação de Pearson.

18.10 Como interpretar o valor de r

O valor de r varia de -1 a 1:

  • 0: Não há efeito (os grupos não diferem).
  • Próximo de 1 ou -1: Efeito muito forte.
  • Sinal positivo ou negativo: Indica a direção da diferença (qual grupo tem valores maiores).

18.10.1 Magnitudes sugeridas (Cohen, 1988; Field, 2013):

Valor de r
0,10 Pequeno (baixo)
0,30 Médio (moderado)
0,50 Grande (alto)

Exemplo:
- r = 0,08 → efeito pequeno
- r = 0,28 → efeito moderado
- r = 0,55 → efeito grande

  • Quanto maior o valor absoluto de r, maior o efeito.
  • O sinal indica a direção da diferença.
  • Sempre interprete o resultado no contexto do seu estudo.
